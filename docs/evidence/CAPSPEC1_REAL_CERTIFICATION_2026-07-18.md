## Reproduction

From a clean checkout on a machine with network access and the repository's
normal Rust/Node test prerequisites:

```bash
scripts/capspec-real-certification.sh
```

Use `--bin path` to certify an already built candidate. The harness binds three
loopback ports, uses `example.com` for the allowed HTTP proof, creates isolated
homes below `target/capspec-real-certification/`, and removes all spawned
processes on exit. It never reads or modifies the operator's personal
`CAPTAIN_HOME`, daemon, Docker state, releases, or external disks.
